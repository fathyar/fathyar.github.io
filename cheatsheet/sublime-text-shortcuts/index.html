Sublime Text Shortcuts

A list of common shortcuts of Sublime Text

Aug 18, 2021  · Tagged: Sublime Text, text editor

Hot keyAction
⌘ + ↹Move to the next tab
⌘ + ⇧ + ↹Move to the previous tab
⌘ + ↑Scroll: one line up
⌘ + ↓Scroll: one line down
⌘ + ⇧ + ↑Shift current line or selected lines: one line up
⌘ + ⇧ + ↓Shift current line or selected lines: one line down
